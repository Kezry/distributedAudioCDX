# Android 配置 App

Android 配置 App 用于在手机或平板上扫描、管理和配置局域网内的 Android 声音端。

## 核心职责

- 扫描局域网内的声音端。
- 展示设备识别码、别名、模式、延迟和连接状态。
- 修改声音端别名。
- 切换声音端运行模式。
- 配置 DLNA 单机或多机模式。
- 配置多机模式下的声道角色。
- 配置手动延迟补偿。

## 配置模式

### 电脑声卡模式

声音端接收 Windows 发送端的低延迟音频流，适合系统音频输出和多机同步。

### DLNA 单机模式

声音端作为普通 DLNA Renderer 使用，适合兼容普通 DLNA 控制器。

### DLNA 多机模式

声音端参与多设备声道播放。发现和控制可兼容 DLNA，但严格同步需要使用项目自定义同步协议。

## 当前实现

- UDP 扫描声音端。
- 列表展示设备别名、识别码、IP 和模式。
- 支持修改别名、模式、声道、延迟档位和手动延迟补偿。
- 通过 TCP 控制协议向声音端发送 `dacdx.configure`。

## MVP 优先级

1. 扫描声音端。
2. 展示设备列表。
3. 修改别名。
4. 切换声卡模式/DLNA 模式。
5. 配置左声道/右声道。
